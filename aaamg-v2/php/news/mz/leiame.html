Bem vindo ao MZn² 2.0 ADV

Agora falta muito pouco para você ter o MZn² instalado e rodando no seu site. Siga o passo-a-passo abaixo para instalar corretamente o sistema.
» Clique aqui e veja o tutorial completo de instalação do MZn².

Passo 1 - Onde colocar?

O MZn² só pode ser executado em servidores que suportam a tecnologia PHP. Muitos hosts pagos atualmente possuem uma versão atualizada do PHP sendo executada, mas a maioria dos hosts gratuitos não possuem.
» Para ver uma lista de hosts gratuitos clique aqui e visite o site do MZn².

Passo 2 - Como colocar?

Grande parte dos hosts têm um sistema chamado FTP. Este sistema permite que você envie os seus arquivos para o seu site. Para instalar o MZn² você precisa de um cliente FTP que suporte alteração de permissões dos arquivos (conhecido como CHMOD). Isto pode estar também na tela de propriedades do arquivo.
» Se você não sabe como usar um cliente FTP, clique aqui e veja o tutorial do Leech FTP.

Passo 3 - Instalando

Depois que você enviar todos os arquivos, basta entrar na URL (endereço) que você instalou o MZn² (ex: http://www.seusite.com.br/mznews/) que você será redirecionado para a instalação. Apenas colocando os dados corretamente, o sistema será instalado está pronto para ser usado!

Passo 4 - Colocando as notícias

Assim que instalado, você pode pegar o código PHP para as notícias na seção Ferramentas > Gerador de PHP. Nele você encontra instruções fáceis para colocar as notícias no seu site.

Ainda com dúvidas?

Se você não consegue instalar o MZn² ou está com alguma dúvida no seu uso, assine o Suporte Técnico do MZn² que nós iremos esclarecer pessoalmente.
» Clique para obter mais informações sobre o Suporte Técnico.
